# Smart Password Strength Checker

This project is a Python-based password strength checker.

It checks password strength based on:
- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Technologies Used
Python
Regular Expressions (re)

## How it Works
The program takes a password as input and evaluates it based on multiple security conditions. It then returns whether the password is Weak, Medium, or Strong.

## Example
Input: Hello@123  
Output: Strong Password
